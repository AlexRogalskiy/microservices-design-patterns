## Java 8 and Spring Boot

### Prerequisites
 * JDK 1.8
 * Maven

### Install

`mvn clean package`

### Run

`mvn spring-boot:run` and access http://localhost:8080

Default users are:

```
admin/password

master/password123

test/test
```

### Swagger
[Swagger](http://localhost:8080/swagger-ui.html)
