## Java 8 and Spring Boot

### Prerequisites
 * JDK 1.8
 * Maven

### Install

`mvn clean package`

### Run Spring Boot

`mvn spring-boot:run` and access http://localhost:8080

Default users are:

```
admin/password

master/password123

test/test
```

### Run React Web app

```
cd react-webapp

sudo npm install

sudo npm start
```

### Swagger
[Swagger](http://localhost:8080/swagger-ui.html)
